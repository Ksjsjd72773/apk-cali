# Calculator - Kivy App

حاسبة بسيطة مبينة بـ Kivy

## 🚀 بناء APK تلقائي

ارفع الملفات على GitHub والـ APK يتبنى تلقائي!

### الخطوات:
1. أنشئ حساب على github.com
2. اضغط **New Repository**
3. ارفع كل الملفات
4. روح **Actions** من القائمة
5. انتظر ينتهي البناء
6. حمّل ملف APK من **Artifacts**

## 📱 تشغيل على الكمبيوتر

```bash
pip install kivy
python main.py
```
